## TDD 연습용 과제 
## 🚀 기능 요구 사항

기본적으로 딜러와 플레이어가 **카드를 뽑아 21에 최대한 가깝게 만드는 게임**이다.
단, **21을 초과하면 Bust(버스트)** 로 즉시 패배한다.

* 게임은 **플레이어 1명 vs 딜러 1명** 으로 진행된다.
* 사용 카드는 일반적인 **52장 트럼프 덱**을 사용한다.

    * 무늬(Suit)는 점수에 영향을 주지 않는다.
    * 카드 랭크(Rank)와 점수는 다음과 같다.

        * 숫자 카드(2 ~ 10): 적힌 숫자만큼 점수
        * J, Q, K: 10점
        * A: **1점 또는 11점**
          → 해당 핸드에서 **21을 넘지 않는 범위에서 점수가 가장 커지도록** 1 또는 11로 계산한다.

---

### 🎮 게임 진행 규칙

1. **게임 시작**

    * 프로그램이 시작되면 다음 문구를 출력한다.

        * `블랙잭 게임을 시작합니다.`
    * 딜러와 플레이어는 각각 **카드 2장씩**을 받는다.
    * 플레이어의 카드는 모두 공개된다.
    * 딜러의 카드는 구현 편의상 **두 장 모두 공개**해도 되고,
      한 장만 공개하고 한 장은 숨긴 것처럼 표현해도 된다. (선택 사항)

2. **플레이어 턴**

    * 플레이어는 자신의 턴에 다음 두 가지 중 하나를 선택할 수 있다.

        * 카드를 한 장 더 받기(Hit)
        * 카드를 더 이상 받지 않고 턴 종료하기(Stand)
    * 카드를 받을 때마다 현재 점수를 다시 계산하고 결과를 출력한다.
    * 플레이어의 점수가 **21점을 초과하면 Bust**이며,
      즉시 플레이어의 패배로 게임이 종료된다.
    * 플레이어는 Bust가 나지 않는 한, **원하는 만큼 여러 번 Hit** 할 수 있다.

3. **딜러 턴**

    * 플레이어가 Bust가 아니고 Stand를 선택한 경우에만 딜러 턴으로 넘어간다.
    * 딜러는 다음 규칙에 따라 **자동으로 행동**한다.

        * 딜러 점수가 **17점 이상**이면 더 이상 카드를 받지 않는다(Stand).
        * 딜러 점수가 **16점 이하**이면 카드를 한 장 더 받는다(Hit).
    * 딜러 점수가 **21점을 초과하면 Bust**이며,
      이 경우 플레이어가 승리한다.

4. **승패 판정**

    * 플레이어와 딜러 모두 Bust가 아닌 경우, 최종 점수를 비교한다.

        * 플레이어 점수 > 딜러 점수 → **플레이어 승리**
        * 플레이어 점수 < 딜러 점수 → **딜러 승리**
        * 플레이어 점수 == 딜러 점수 → **무승부**

5. **게임 재시작 / 종료**

    * 한 판이 끝난 후 다음 문구를 출력한다.

        * `게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.`
    * 입력이 `1`이면 새로운 덱(또는 재사용한 덱)을 준비하여 게임을 다시 시작한다.
    * 입력이 `2`이면 게임을 종료한다.

6. **예외 처리**

    * 플레이어 행동 입력(`Y/N` 등), 재시작/종료 입력(`1/2`)에서
      지정된 값이 아닌 **잘못된 값이 입력되면** `IllegalArgumentException`을 발생시키고
      애플리케이션은 종료되어야 한다.

### 🧾 입출력 요구 사항

#### ✅ 입력

* 플레이어의 행동 입력

    * 카드를 더 받을지 여부 입력
    * 예시: `Y`(또는 `y`) → Hit, `N`(또는 `n`) → Stand
      (정확한 문자열은 구현에서 정하고 **일관되게 사용**한다.)
* 게임이 끝난 뒤 재시작 / 종료 선택

    * `1` → 게임 재시작
    * `2` → 게임 종료

#### ✅ 출력

* 게임 시작 문구

```text
블랙잭 게임을 시작합니다.
```

* 플레이어 및 딜러 카드 / 점수 출력 (예시)

```text
플레이어 카드: [10, 6] - 현재 점수: 16
딜러 카드: [9, ?]
```

* 플레이어 행동 선택 프롬프트

```text
카드를 더 받으시겠습니까? (Y/N)
```

* 플레이어가 카드를 한 장 더 받았을 때 (예시)

```text
플레이어가 카드를 한 장 더 받았습니다.
플레이어 카드: [10, 6, 4] - 현재 점수: 20
```

* 플레이어 Bust 시 (예시)

```text
플레이어 카드: [10, 6, 8] - 현재 점수: 24
플레이어 Bust! 딜러의 승리입니다.
```

* 딜러 턴 진행 예시

```text
딜러의 턴을 시작합니다.
딜러 카드: [9, 7] - 현재 점수: 16
딜러가 카드를 한 장 더 받습니다.
딜러 카드: [9, 7, 2] - 현재 점수: 18
```

* 최종 결과 출력 예시

```text
플레이어 카드: [10, 6, 4] - 최종 점수: 20
딜러 카드: [9, 7, 2] - 최종 점수: 18
플레이어의 승리입니다!
```

또는

```text
플레이어 카드: [10, 7] - 최종 점수: 17
딜러 카드: [9, 8] - 최종 점수: 17
무승부입니다.
```

* 게임 재시작 / 종료 선택 문구

```text
게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.
```
